### ⚠WORK IN PROGRESS⚠

# HelpNow connects people needing help with everyday tasks with people eager to help.



Currently implemented features:
- View posts on map
- Create post
- Edit post
- Volunteer/withdraw volunteer from post
- Sign in/register

For users safety the posts/tasks address is revealed to the volunteer only an hour in advance. 

Made with modern full-stack technologies (graphql, next.js, typescript, Prisma ORM, postgresql, Google Cloud API etc.)
