Frontend: React Next.js Typescript Chakra UI Tailwind
API: GraphQL, Apollo server
Backend: Next.js routing
Database: Postgresql with prisma 
Mobile: Flutter
Hosting: AWS, vercel

Utils:
Input validation: ZOD
Authentication: next auth
Password hashing: bcrypt
Geocoding: Google maps geocode API

FEATURES:
- chat between (potential) volunteers and post creator
- Convert address to coordinates when submitting post
- When not signed in, show "X people need help in your area" and a call to action
